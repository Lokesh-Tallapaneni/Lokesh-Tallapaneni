### About me

I'm a Python developer at Cloudpi who likes building things from first principles rather than gluing libraries together. My recent work spans a from-scratch async web framework, a static-typed Python-to-Rust transpiler, and a pre-registered quant research system. I care about clean internals, precompiled hot paths, and tooling that holds up under real load.

- 🔭 Currently building **Veloce** — an ASGI-native async Python web framework
- 🦀 Exploring compilers & transpilers with **py2rs** (Python → Rust)
- ☁️ AWS Certified Cloud Practitioner
- 🎓 B.Tech in Artificial Intelligence

### 🚀 Featured projects

**[veloce](https://github.com/Lokesh-Tallapaneni/veloce)** — Ultra-fast async Python web framework. ASGI-native, batteries included: a from-scratch radix-tree router, typed dependency injection, Pydantic v2 validation, OpenAPI 3.1, WebSockets, and built-in middleware — not a wrapper around Starlette or FastAPI. `pip install veloceframework`

**[Telegram-Message-Forwarder](https://github.com/Lokesh-Tallapaneni/Telegram-Message-Forwarder)** — Tool to forward and route messages across Telegram channels and chats.

**[Amazon-Affiliate-Dashboard](https://github.com/Lokesh-Tallapaneni/Amazon-Affiliate-Dashboard)** — Dashboard for tracking and analyzing Amazon affiliate performance.

**[ChannelActionsBot](https://github.com/Lokesh-Tallapaneni/ChannelActionsBot)** — Telegram bot to auto-approve chat join requests.
